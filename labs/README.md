# Labs

Hands-on labs, one folder per Azure service. Each lab is self-contained and
documented so it can be re-run from scratch.

## How to Use

- Each lab folder contains a `README.md` following the
  [lab documentation template](TEMPLATE.md).
- Work labs roughly in numeric order; later labs may build on earlier ones.
- Link each lab back to the week it was completed in [`../notes/`](../notes/) and
  to the objectives it covers in [`../docs/exam-objectives.md`](../docs/exam-objectives.md).
