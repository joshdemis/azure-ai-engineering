# Projects

End-to-end projects that combine multiple Azure services into cohesive, portfolio-
grade solutions. Where labs isolate a single service, projects integrate them.

## How to Use

- Each project folder has a `README.md` following the
  [project documentation template](TEMPLATE.md).
- Projects should reference the labs and patterns they build on.
- Infrastructure for projects should live under [`../infra/`](../infra/).
