# Notes

Weekly learning notes captured throughout the journey. One file per week keeps
progress visible and searchable over time.

## Conventions

- One file per week: `week-01.md`, `week-02.md`, `week-03.md`, …
- Start each week from the [weekly note template](TEMPLATE.md).
- Link notes to their matching lab under [`../labs/`](../labs/) and to the
  relevant week in [`../docs/study-plan.md`](../docs/study-plan.md).
