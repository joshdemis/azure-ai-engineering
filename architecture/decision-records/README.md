# Architecture Decision Records (ADRs)

A log of significant, hard-to-reverse decisions and the context behind them.
ADRs make reasoning durable — future-you will thank present-you.

## Conventions

- One decision per file: `NNNN-short-title.md` (e.g. `0001-use-bicep-for-iac.md`).
- Number sequentially from `0001`.
- Start from [`TEMPLATE.md`](TEMPLATE.md).
- Decisions are immutable once **Accepted**. To change one, add a new ADR that
  **supersedes** it and cross-link both.

## Statuses

`Proposed` → `Accepted` → (`Superseded` | `Deprecated`)
