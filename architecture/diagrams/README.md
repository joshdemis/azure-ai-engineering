# Diagrams

Architecture and flow diagrams for labs and projects.

## Conventions

- Store the **editable source** here (e.g. `.drawio`, `.excalidraw`, `.mmd`).
- Export images (`.png`, `.svg`) to [`../../assets/diagrams/`](../../assets/diagrams/)
  for embedding in Markdown.
- Name files descriptively: `<area>-<subject>.<ext>` (e.g. `rag-ingestion-flow.drawio`).
