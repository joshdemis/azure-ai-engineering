# Assets

Static assets referenced from documentation across the repository.

## Structure

| Path | Purpose |
| ---- | ------- |
| [`images/`](images/) | Screenshots, badges, and general images. |
| [`diagrams/`](diagrams/) | Exported diagram images (source lives in [`../architecture/diagrams/`](../architecture/diagrams/)). |

## Conventions

- Use descriptive, kebab-case filenames.
- Prefer `.svg` for diagrams and `.png` for screenshots.
- Keep file sizes reasonable; optimise images before committing.
