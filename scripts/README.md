# Scripts

Helper and automation scripts supporting labs, projects, and repository upkeep.

## Conventions

- Keep scripts small, single-purpose, and idempotent where possible.
- Document each script's purpose, inputs, and prerequisites in a header comment.
- **Never** hard-code secrets — read them from environment variables or Key Vault.
- Prefer parameterised scripts over copy-paste variants.
